# MyTinyAndroidApp

Eine kleine Android-Anwendung mit SSO-Integration.

## Features

- Android-App mit Single Sign-On (SSO) Funktionalität
- Keycloak Integration für Authentifizierung
- Moderne Android-Architektur mit Dagger Hilt
